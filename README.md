## Spam Email Detection 

This repository contains a solution for an automated spam email detection task. The objective is to build a machine learning model that classifies emails as spam or not spam based on their content.

The full problem statement and dataset are available on the official competition platform:  
[View Problem & Dataset](https://platform.olimpiada-ai.ro/ro/problems/58)

Due to size limitations, the dataset is not included in this repository. Please download it from the official source.

## Task

Given labeled training data, the goal is to train a model that can accurately predict whether an email is spam. Predictions are generated for unseen data and evaluated based on accuracy.
